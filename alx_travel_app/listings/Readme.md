# ALX Travel App 0x00

**ALX Travel App** is a web application that allows users to explore travel destinations, manage bookings, and interact with travel-related data. This project is built using **Django** and **SQLite** as the database, providing a simple and efficient backend for travel management.

## Features

- User authentication and profile management  
- CRUD operations for travel destinations  
- Booking system for trips  
- RESTful API endpoints for integration  
- Nested relationships for conversations and messages  
- Admin panel for managing data  

---

## Technologies

- **Backend:** Django 5.x  
- **Database:** MySQL 
- **Language:** Python 3.x  
- **API:** Django REST Framework (DRF)  
- **Others:** Environ for environment variables
